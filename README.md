# Jokenpô Web

Um jogo de Pedra, Papel e Tesoura interativo para jogar diretamente no seu navegador.

## Como Jogar

1.  Abra o arquivo `index.html` em seu navegador web.
2.  Você verá o título do jogo e três botões: Pedra, Papel e Tesoura.
3.  Clique no botão correspondente à sua escolha.
4.  O resultado da rodada (sua escolha, a escolha do computador e quem venceu) será exibido na tela, juntamente com a pontuação atual.

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## Status do Projeto

Este é um projeto simples e funcional de Jokenpô. Sinta-se à vontade para explorar o código e contribuir!
